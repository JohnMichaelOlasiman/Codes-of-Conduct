# Professional Codes of Conduct in Computing

A full-screen, presentation-style Next.js deck for classroom reporting.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Controls

- Mouse wheel or trackpad
- Arrow keys, Page Up, Page Down, Home, and End
- On-screen previous/next controls
- Slide navigation dots
- Swipe up/down on touch screens

## Structure

```text
app/
  globals.css
  layout.tsx
  page.tsx
components/
  icons.tsx
  NavigationDots.tsx
  Presentation.tsx
  ProgressIndicator.tsx
  Slide.tsx
```
